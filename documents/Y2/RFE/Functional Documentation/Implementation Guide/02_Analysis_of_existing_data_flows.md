## Analysis of data sources

Analysis of data sources

By SQL query :

To retrieve the listof scheduled data provenances, run a query like the following query on the SQL Y2 Monitor:

File name: Analyse_Provenances_De_Donnees.sql

Contents

-- 2022-08-02 -- Requete pour extraire les provenances de données planifiées via une tache active SELECT SKJ_JOBID, SKJ_LIBELLE, SKJ_ACTIF, SKJ_LIBELLEAUTO, SKJ_OCCURENCE, SKJ_TYPEJOB, SKJ_DATECREATION, SKJ_DATEMODIF, SKJ_CREATEUR, SKJ_UTILISATEUR, MPO_TYPEREPRISE, MPO_LIBELLE, MPO_REPENTREE, MPO_REPREJET, MPO_REPARCH, MPO_NOMFICLOG, MPO_HORODATAGEFIC FROM STKJOBS LEFT JOIN STKDETAILS ON SKD_JOBID=SKJ_JOBID LEFT JOIN MEAIPROVENANCE ON SUBSTRING(SKD_DATA,CHARINDEX("Value",SKD_DATA)+7,CHARINDEX("Type",SKD_DATA,CHARINDEX("Value",SKD_DATA)+7)-2 - (CHARINDEX("Value",SKD_DATA)+7)) LIKE "%" + MPO_TYPEREPRISE + "%" WHERE SKD_EXENAME="cgimode" AND SKJ_ACTIF="X" AND SKJ_OCCURENCE>0 AND SUBSTRING(SKD_DATA,CHARINDEX("Name",SKD_DATA,CHARINDEX("Name",SKD_DATA)+1),17) LIKE "%PROVENANCE%"
