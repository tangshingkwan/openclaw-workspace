# Y2Plugin TaxEngine V26

## Overview

Plugin for tax calculation in Cegid Retail Y2 v26. Alternative to external tax connectors (Avalara, Vertex).

## Key Takeaways

### Core Methods
- `CalculateTax` — Calculate tax for transaction
- `GetTaxRates` — Get applicable tax rates
- `GetTaxRules` — Get tax rules by jurisdiction

### Use Cases
- POS tax calculation
- Invoice tax computation
- Multi-jurisdiction tax handling
