# Y2Plugin Audit V26

## Overview

Plugin for retrieving audit and operational data in Cegid Retail Y2 v26 — store info, users, POS terminals, inventory tracking.

### Usage Methods

| Method | Purpose | Status |
|--------|---------|--------|
| `GetStores` | List all stores | Released |
