# Y2 Finance & Controlling V26

## Overview

Finance and controlling in Cegid Retail Y2 v26. Covers accounting closure, inventory valuation, and fiscal year management.

**Source:** Extracted 2026-02-27 | **Version:** Y2 v26

## Accounting Closure

### Purpose

- Streamline financial reporting
- Inventory valuation on specific date
- Handle non-received supplier invoices
- Block Back Office operations post-closure

### Key Points

- Based on effective document date
- Once closed, prevents document creation/modification before closure date
- Manages non-received invoices (not accrued expenses/deferred income)

### Required Settings

| Setting | Location |
|---------|----------|
| Accounting journals | Back Office > Administration > Link to Accounting > Journals |
| General accounts | Configuration |
| Company settings | Document config |
| Store record | Config |
| Access rights | User permissions |

## Configuration Steps

1. Create accounting journals
2. Create general accounts
3. Configure company settings
4. Configure documents
5. Configure store record
6. Manage access rights
