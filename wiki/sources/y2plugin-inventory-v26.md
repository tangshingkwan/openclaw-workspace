# Y2Plugin Inventory V26

## Overview

Plugin for inventory queries and stock management in Cegid Retail Y2 v26.

## Key Takeaways

### Core Methods
- `GetStock` — Get stock levels by warehouse
- `GetStockByItem` — Get stock for specific item
- `GetStockByStore` — Get stock by store/location

### Use Cases
- Real-time stock queries
- Inventory synchronization
- Low stock alerts
