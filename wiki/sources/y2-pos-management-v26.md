# Y2 Point-of-Sale Management V26

## Overview

Front Office (POS) management in Cegid Retail Y2 v26. Covers register configuration, checkout screens, and point-of-sale operations.

**Source:** Extracted 2026-02-27 | **Version:** Y2 v26

## Register Settings

Configurable via Back Office or Front Office at:
- `Back Office > Settings > Front Office > Register`
- `Front Office > Settings > Registers > Registers`

### Register Record Tabs

| Tab | Purpose |
|-----|---------|
| General | Register identification |
| Preferences | Touch screen & keyboard config |
| Additions | Web page buttons, input config |
| Management | Transaction management |
| Discount | Discount rules |
| Daily operations | Daily workflow settings |
| End of day | Closing procedures |
| Cash float | Cash handling |
| Layout | Checkout screen layout |
| Cash register receipt | Receipt formatting |
| Peripherals | Hardware configuration |
| Communications | Network/communication settings |
| Standalone mode | Offline operation |
| Services | Service configuration |

## Key Features

- Touch screen and keyboard configuration
- Web page display via touch pad buttons
- Receipt input configuration
- Peripheral device setup
- Standalone/offline mode support
