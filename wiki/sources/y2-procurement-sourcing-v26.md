# Y2 Procurement & Sourcing Management V26

## Overview

Procurement and sourcing management in Cegid Retail Y2 v26. Covers supplier records, purchase processes, and sourcing workflows.

**Source:** Extracted 2026-02-27 | **Version:** Y2 v26

## Supplier Management

### Supplier Record

Essential basic data — no purchases without associated supplier. Controls:
- Document behavior in entry mode
- Barcode assignment
- Item referencing
- Default values

### Settings

- **Company:** Auto supplier codes, distinct barcodes per supplier
- **Search priorities:** Up to 6 configurable priorities
- **Field settings:** User-defined display fields

### Access Rights

| Menu | Purpose |
|------|---------|
| Menu 26 | Display/create/modify suppliers |
| Menu 105 | Settings tables |
| Menu 106 | Event log & purge |
| Menu 110 | Basic data access |
