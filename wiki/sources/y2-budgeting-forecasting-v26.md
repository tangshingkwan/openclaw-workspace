# Y2 Budgeting & Forecasting V26

## Overview

Sales objectives and forecasting module in Cegid Retail Y2 v26. Track sales forecasts against actuals using history-based objectives.

**Source:** Extracted 2026-02-27 | **Version:** Y2 v26

## Key Features

- Establish and track sales forecasts
- Define objectives based on sales history
- Three objective entry methods:
  - Evolution percentage
  - Evolution amount
  - Direct objective value
- Real-time tracking in store
- Reporting via cubes and dashboards

## Setup Areas

| Area | Purpose |
|------|---------|
| General settings | Serialization, company settings |
| Access rights | User group permissions |
| Session creation | General and application config |
| Allocation schedule | Define how objectives are distributed |
| Objectives definition | Entry, checking, adjustment |
| Schedule | Daily goals management |

## Reporting

- Cube reporting
- Dashboard views
- Store goals tracking
- Objectives by period analysis
