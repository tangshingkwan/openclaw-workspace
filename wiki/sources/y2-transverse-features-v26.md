# Y2 Transverse Features V26

## Overview

Getting started with Cegid Retail Y2: connection, login sequence, and homepage.

**Source:** Extracted 2026-02-27 | **Version:** Y2 v26

## Login Sequence

Messages appear in order:
1. Initialization
2. Loading database structure
3. Loading dictionaries
4. Checking licenses
5. Loading menus
6. Checking database structure
7. Loading data to cache
8. Loading user profile
9. Compiling scripts
10. Loading favorites
11. Completing connection

## Access

- Double-click Back Office or Front Office icon
- Or via Windows Start menu

## Homepage

Displayed after login if no favorites defined. Use Maximize/Reduce menu bar button to access application menus.
