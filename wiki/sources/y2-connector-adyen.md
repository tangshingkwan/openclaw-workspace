# Y2 Adyen Connector

## Overview

Adyen PayByLink connector for Cegid Retail Y2 LiveStore and MPOS. Enables payment processing through Adyen's PayByLink system.

**Release Date:** February 20, 2025 | **Delivered with package:** 24.0.3 | **Version:** 2.1.1

## Key Takeaways

### Features
- **PayByLink** — Generate payment links for customer checkout
- **LiveStore integration** — Web-based POS with Adyen payments
- **MPOS integration** — Mobile POS with Adyen payments
- No breaking changes currently

### Use Cases
- E-commerce payment links
- In-store contactless payments
- Mobile payment processing
