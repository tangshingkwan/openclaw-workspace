# Y2 v26 - Omni-Channel Management

## Overview

Omni-channel management guide for Cegid Retail Y2 v26. Covers store/warehouse configuration, multi-warehouse management, and e-commerce integration.

## Key Takeaways

### Store Record Tabs
- **Contact information** — Address, phone, hours
- **Information** — Store type, category
- **Third-party** — Legal entity, tax info
- **Linked warehouses** — Multiple warehouses per store
- **Accounting** — Financial settings
- **Omni-channel** — E-commerce, pickup points
- **Replenishment** — Stock replenishment rules
- **Delivered/picked up** — Fulfillment options

### Warehouse Modes
- **Single warehouse** — One warehouse = one store (default)
- **Multi-warehouse** — Multiple warehouses per store (irreversible conversion!)

### Omni-Channel Features
- **E-commerce integration** — Online order management
- **Click & collect** — Buy online, pickup in store
- **Ship from store** — Drop-shipping from retail locations
- **Pickup points** — Alternative delivery locations

### Key Configurations
- User restrictions by store type
- Inter-warehouse transfers
- Stock replenishment automation
