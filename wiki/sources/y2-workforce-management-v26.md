# Y2 Workforce Management V26

## Overview

Staff discount and clothing allowance management in Cegid Retail Y2 v26. Manages non-standard price lists for employees.

**Source:** Extracted 2026-02-27 | **Version:** Y2 v26

## Use Cases

| Program | Description |
|---------|-------------|
| Clothing | Staff entitled to clothing from stock up to specific amount |
| VIP | Special price lists for staff/customers |
| Staff allowance | % discount with max annual amount |
| Executive | 100% discount with various limits |

## Example Configurations

- **Example 1:** 30-50% discount based on brand, 6 months seniority required
- **Example 2:** Max €1,500/year, 30% discount, regional manager approval
- **Example 3:** 100% discount, 10 items per category per season

## Settings

| Area | Purpose |
|------|---------|
| Program categories | Define discount types |
| Access rights | User group permissions |
| User restrictions | Store-based limits |
| Special conditions | Characteristics, pricing, controls |

## Customer Linking

- Employees defined as customers
- Program history tracking
- Batch program creation/closing
- Import with allowances
