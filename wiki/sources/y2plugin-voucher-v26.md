# Y2Plugin Voucher V26

## Overview

Plugin for managing vouchers and gift cards in Cegid Retail Y2 v26.

**Registration:** January 9, 2026 | **Plugin Build:** #6.96

## Key Takeaways

### Core Methods

| Method | Purpose | Status |
|--------|---------|--------|
| `Create` | Create a new voucher | Released |
| `CancelCreate` | Cancel voucher creation | Released |
| `Redeem` | Redeem a voucher | Released |
| `CancelRedeem` | Cancel voucher redemption | Released |
| `Update` | Update voucher details | Released |
| `Cashout` | Cash out voucher balance | Released |

### Use Cases
- Gift card issuance and redemption
- Loyalty points/coupons
- Store credits
