# Y2 Localization V26

## Overview

Translation and localization management in Cegid Retail Y2 v26. Full translation process for interfaces, screens, and reports.

**Source:** Extracted 2026-02-27 | **Version:** Y2 v26

## Key Principles

- FRA (French) terms are the reference for all translations
- French dictionary (YYLANG1036) must always be installed
- Term customization preserved during updates
- No dictionary volume limits (only memory constraints)

## Translation Tables

| Table | Purpose |
|-------|---------|
| YYLANGPAYSISO | Country/language mapping |
| YYTRADTERM | Terminology translations |
| YYTRADITEM | Item label translations |
| YYTRADGUI | GUI element translations |

## What Can Be Translated

- Interface and screens
- Connection screen
- Reports
- Language-specific characters
- User-defined report names
- Receipt names
