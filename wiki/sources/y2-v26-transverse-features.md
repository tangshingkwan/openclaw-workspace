# Y2 v26 - Transverse Features

## Overview

Core user interface and navigation guide for Cegid Retail Y2 v26. Covers login process, menus, favorites, and working environment.

## Key Takeaways

### Login Sequence
1. Initialization
2. Loading database structure
3. Loading dictionaries
4. Checking licenses
5. Loading menus
6. Checking database structure
7. Loading data to cache
8. Loading user profile
9. Compiling scripts
10. Loading favorites
11. Completing connection

### Homepage Features
- **Favorites** — Up to 10 customizable shortcuts (new in v20)
- **My last functions** — History of last 5 used functions
- **Module switching** — Change between Back Office / Front Office
- **Menu pinning** — Keep menu bar always visible

### Working Environment
- **Module structure** — Menus > Commands > Functions
- **Access rights** — Serialized options control menu visibility
- **Customizable fields** — User-defined field displays

### Navigation
- Double-click icons or use Windows Start menu
- Maximize/Reduce menu bar toggle
- Favorites auto-remove if access rights revoked
